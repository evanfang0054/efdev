# 文档规范

项目文档建议使用[dumi](https://d.umijs.org/)进行编写

## 文案风格

- 多检查，确保没有错别字
- 言简意赅
- 注意特殊名词的大小写：AJAX、Highcharts、React、JavaScript、HTML，无论是否在句首都应该以同样的方式写（有 lint 工具限制）
- 只有中文或中英文混排中，一律使用中文/全角标点

## 文档格式

严格遵循[markdownlint](https://github.com/DavidAnson/markdownlint)格式
